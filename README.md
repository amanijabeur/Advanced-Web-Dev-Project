# Advanced Web Dev Project

A personal website built with React and Vite for my Advanced Web Development course at TBS for the academic year 2024/2025.

## Features

- Personal intro / About Me section
- Story search: filter a list of stories in real time by title

## Tech Stack

- React 18
- Vite
- CSS (no external UI library)

## Getting Started

```bash
cd React
npm install
npm run dev
Then open http://localhost:5173 in your browser.
